# Notebooks (optional)

Jupyter notebooks are **optional** in this course. They are useful for exploration, plotting, and prototyping parsers.

## Policy

- Reproducible extraction, cleaning, building, and validation must live in `scripts/`.
- Anything required for grading or publication should run without opening a notebook.
- If you use a notebook, copy finalized logic into the appropriate script and cite the notebook path in your report only if needed.

Do not submit notebooks as the sole implementation of required pipeline steps.
